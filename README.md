# BP Tracker 🫀

**Your health should never depend on politics.**

In 2024–2025, many health apps — including Omron's official app — became 
unavailable in Russia due to geo-restrictions and sanctions. But blood 
pressure monitoring is not a luxury. It's a daily necessity for millions 
of people.

So I built this.

BP Tracker is a fully open-source iOS app that lets you scan blood pressure 
readings from **any monitor** using your phone camera, track your history, 
and visualize trends — no accounts, no cloud, no subscriptions, no politics.

## Features

- 📷 **Camera OCR** — point your phone at any BP monitor display, readings 
  are detected automatically using Apple's Vision framework
- 📊 **Trends chart** — visualize systolic/diastolic changes over time
- 💾 **100% local** — all data stored on your device via SwiftData
- 🔓 **Open source** — fork it, improve it, make it yours

## Why Open Source?

Health data is personal. Health tools should be accessible to everyone, 
regardless of where they live or what political decisions are made around them.
This project is my small contribution to that idea.

## Tech Stack

- **SwiftUI** — declarative UI
- **SwiftData** — local persistence
- **Apple Vision framework** — on-device OCR, no internet required
- **Swift Charts** — native data visualization
